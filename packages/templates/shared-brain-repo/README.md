# Shared Brain Repo Template

The concrete v1 seed is generated by `packages/brainctl/src/main.ts` so profile values can be injected deterministically.

Required directories in generated repos:

- `wiki/`
- `raw/`
- `proposals/`
- `logs/`
- `manifests/`
- `derived/` ignored by git

The generated repo contract keeps canonical markdown/manifests/raw artifacts in git and keeps derived indexes local.
